# Fury-6: Advanced All-Terrain Hexapod 🐉🤖

**Fury-6** is a high-performance biomimetic hexapedal robotic platform designed for autonomous exploration and search-and-rescue missions in unstructured environments.

## 📖 Project Abstract
Fury-6 is engineered to venture where humans and traditional wheeled vehicles cannot. Utilizing a bio-inspired six-legged architecture, the system achieves superior stability through complex **Inverse Kinematics (IK)** algorithms. 

The design philosophy is profoundly **inspired by the agility, intelligence, and protective nature of "Toothless" (the Night Fury)**, aiming to redefine the standards of robotic intervention in global crisis management.

## 🚀 Key Features
* **Terrain Mastery:** Navigates extreme terrains, from disaster debris to unstable mountain slopes.
* **18 Degrees of Freedom (DOF):** Mimics adaptive natural gait cycles for precise locomotion.
* **Search & Rescue (SAR):** Optimized for deep exploration and evacuation support in hazardous zones.
* **Web-Integrated Control:** Designed to be managed via a professional web interface (Bogatiy Robotics).

## 🛠 Technical Stack
* **Hardware:** ESP32 (Main Controller), PCA9685 (Servo Driver), MG90S Metal Gear Servos.
* **Software:** C++, PlatformIO, Tailwind CSS (Web UI).
* **Design:** Fusion 360 (CAD), Unity (Simulation).

## 📂 Project Structure
* `/cad`: 3D models and STL files for Fury-6 chassis and legs.
* `/firmware`: ESP32 source code and IK algorithms.
* `/web-ui`: Dashboard for remote monitoring and control.

## 🛡 Vision: The Peace Journey
More than just a machine, **Fury-6** represents a "Peace Journey"—using advanced robotics not for conflict, but for the preservation and protection of human life.
